## İstifadə

1. `EPM-310-latest.exe` faylını işlədin
2. Brauzer avtomatik açılır
3. Əyarlardan COM port və Baud Rate seçin
4. **Bağlan** düyməsinə basın

## Tələblər

- Windows 7 / 10 / 11 (64-bit)
- USB-RS485 adapter (CH340)
- EPM-310 cihazı
